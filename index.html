📊 Google Sheets Integration

Your WHOOP data auto-exports to data/whoop.csv daily. Import it into Google Sheets in 2 ways:

Option A — One-click import (recommended)
In Google Sheets: File → Import → Upload → whoop.csv
Download: whoop.csv

Option B — Live auto-sync with Apps Script
In your Sheet: Extensions → Apps Script, paste:

function importWhoop() {
  const url = 'https://dacameragirl.github.io/whoop-dashboard/data/whoop.csv';
  const csv = UrlFetchApp.fetch(url).getContentText();
  const rows = Utilities.parseCsv(csv);
  SpreadsheetApp.getActiveSheet()
    .getRange(1, 1, rows.length, rows[0].length)
    .setValues(rows);
}
// Run importWhoop() on a time trigger for auto-sync!

Set a time-driven trigger in Apps Script to auto-pull daily. Your dashboard CSV updates automatically, so your Sheet stays in sync. 🔄
